## Installation

Install Crystal:

```bash
# OS X
brew update
brew install crystal-lang

# Debian / Ubuntu
curl http://dist.crystal-lang.org/apt/setup.sh | sudo bash
sudo apt-get install crystal
```

Install dependencies:

```bash
cd crystal-wholesaler-edi
shards install
```
